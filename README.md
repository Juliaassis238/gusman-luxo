# Dr. André Gusman — Cirurgia Oral & Urgência

Site profissional para **Dr. André Gusman**, cirurgião-dentista CRO 2038, localizado em Sorocaba, SP.

## 📋 Sobre

Este é um site responsivo e moderno que apresenta os serviços odontológicos oferecidos pelo Dr. André Gusman, incluindo:

- **Cirurgia Oral** — Procedimentos cirúrgicos especializados
- **Endodontia** — Tratamento de canal
- **Odontopediatria** — Atendimento infantil
- **Urgência Odontológica 24h** — Atendimento de emergência
- **Próteses Dentárias** — Reabilitação protética
- **Clareamento Dental** — Estética dental
- **Restaurações** — Restaurações estéticas e funcionais
- **Limpeza e Manutenção** — Higiene bucal

## 🎨 Características

- Design elegante e moderno com paleta de cores sofisticada (dourado, preto e tons neutros)
- Seção de "Antes e Depois" com slider comparativo
- Formulário de agendamento
- Totalmente responsivo (mobile, tablet, desktop)
- Otimizado para SEO com schema.org
- Animações suaves e transições elegantes
- Performance otimizada

## 🚀 Como Usar

1. Clone este repositório:
```bash
git clone https://github.com/seu-usuario/site-andre-gusman.git
```

2. Abra o arquivo `index.html` em um navegador web

3. Para servir localmente com um servidor HTTP:
```bash
# Python 3
python3 -m http.server 8000

# Node.js (com http-server)
npx http-server
```
